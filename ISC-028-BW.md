# ⚠️ Special branch: `inkbird-isc028bw`

This Git branch is **only** for the **Inkbird ISC-028-BW** smoker in Home Assistant.

It is **not** the official [make-all/tuya-local](https://github.com/make-all/tuya-local) release.

## Who should install this?

- You have an **ISC-028-BW** and need climate, smoker switch, and DP 102 to work locally.

## Who should **not** use this as their only Tuya Local?

- Everyone else — stay on **make-all/tuya-local** from HACS.
- ISC-028-BW users who prefer **minimal patch**: keep official HACS + copy 4 files (see docs).
